Adjustment Pengambilan Paket

Fitur Adjustment digunakan hanya jika informasi Sisa paket di MIS lebih banyak dari kondisi sesungguhnya.

Contoh:

  • Pada MIS customer A memiliki paket “Facial X 5x” dengan Sisa 2, namun ternyata di buku paket (kondisi sesungguhnya) sisa paket customer A adalah hanya 1. Maka lakukan Adjustment untuk menyesuaikan selisih 1 pengambilan paket tsb.


Catatan:

  • Pengambilan paket Adjustment tidak dihitung sebagai Net Sales.
  • Pengambilan paket Adjustment tidak mengurangi pemakaian bahan (menyusul).
  • Pengambilan paket Adjustment hanya bisa dilakukan oleh Accounting.


Cara pakai:

  • Lakukan Ambil Paket terlebih dahulu seperti biasa.
  • Klik pada Daftar Pengambilan Paket ybs, lalu klik tombol Adjustment.
