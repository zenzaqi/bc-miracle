Troubleshoot Gammu

  • Untuk melakukan troubleshooting program gammu yang mungkin bermasalah, kita memerlukan program tambahan bernama WinTail.


  • Download WinTail.
    Note:

       Anda dapat mendownload WinTail pada komputer server dengan alamat folder share:
               \\192.168.1.2\Source\SMS Gateway\WinTail.exe


  • Program WinTail.
    Saat program wintail dijalankan, pertama kita akan ditanyakan untuk membuka salah satu file. Arahkan ke file log dari gammu (opsi log pada file konfigurasi smsdrc harus dalam keadaan aktif) seperti gambar di bawah ini:
