Email Blast

Master Data Customer dapat digunakan apabila kita ingin mengirimkan Email Blast, yaitu mengirimkan email secara bersamaan ke banyak customer.


Fungsi ini adalah dengan memanfaatkan fitur Export Excel yang ada di setiap modul MIS. Untuk lebih jelasnya juga dapat dilihat langkah-langkah di bawah ini:


  • Sebelum melakukan Export Excel, pilih dahulu customer yang akan dicetak melalui Adv Search (atau Simple Search)




  • Setelah data customer yang diinginkan muncul, klik tombol Export Excel


       
       

  • Akan muncul tampilan seperti gambar di bawah ini :


       

  • Klik Open with untuk melihat saja (Read-only), Klik Save File untuk menyimpan file tersebut.


  • Jika File Excel tersebut dibuka akan seperti pada gambar di bawah ini :


       

       

  • Copy kolom email, dan Paste-kan ke MS Outlook atau program sejenisnya yang digunakan untuk mengirim Email Blast


Note:

  • Pencarian yg dilakukan sebaiknya jangan terlalu banyak, karena dapat menyebabkan error memory tidak mencukupi (bisa dibatasi dengan No Cust Awal dan No Cust Akhir pada Adv Search)
