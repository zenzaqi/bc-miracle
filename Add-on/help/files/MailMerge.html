Mail Merge - MS Office 2003

Mail Merge adalah salah satu fitur dalam Microsoft Word untuk mangambil data secara otomatis dari database, tanpa kita perlu mengetikkannya satu per satu.

Data yang diambil bisa berupa nama, alamat, kota, telp, dll milik customer yg terdaftar di database MIS.


Berikut salah satu contoh untuk melakukan Mail Merge, dengan menggunakan database customer di MIS:


  1. Ambil data customer dari MIS dengan menggunakan Export Excel:
    • Buka Master Data Customer (menu Master Data --> Customer)
    • Cari kriteria customer yg diinginkan dengan menggunakan Adv Search
    • klik Export Excel



  1. Simpan hasil Export di folder tertentu dan buka file hasil export tersebut maka akan muncul seperti gambar di bawah ini.




  1. Rapikan terlebih dahulu data mana saja yg akan ditampilkan di MS Word dan beri header diatas setiap filednya.

       Contoh : ambil data Nama, Alamat, dan Kota. Maka akan menjadi seperti ini :



  1. Simpan file tersebut.


  1. Buka MS Word (contoh yg digunakan disini adalah MS Word 2003). Contoh: membuat surat pemberitahuan kepada customer.


  1. Aktifkan Mail Merge dengan cara: Klik Tools - Letters and Mailings - Show Mail Merge Toolbars. Maka Toolbars untuk Mail Merge akan nampak.




  1. Klik Open Data Source dan cari File Excel yg tadi sudah disimpan (customer.xls)




  1. Maka akan keluar Select Table - Klik OK
  2. Tekan Insert Merge Fields
